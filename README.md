# VEHIREN (VEHICLE RENTAL SYSTEM)
The aim of this project was to create an online platform for renting scooters and bikes, providing users with a seamless and secure booking experience
Functionality:
The application allows users to search for vehicles based on fuel type, color, and name. They can view detailed descriptions and images of the bikes, and book them for desired periods based on availability. The admin panel enables administrators to verify user-submitted documents and grant access to rent the vehicles.
